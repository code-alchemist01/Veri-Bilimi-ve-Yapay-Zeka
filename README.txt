## 🚀 Getting Started

To embark on your Python journey, you'll need:

1. 🐍 Python 3.x installed on your system (We use Python 3.11.9)
2. 📓 Jupyter Notebook or JupyterLab

Install Jupyter easily with pip:

""" pip install jupyter """


## 🏃‍♂️ Running the Notebooks

1. 📥 Clone this repository to your local machine
2. 📂 Navigate to the repository folder in your terminal
3. 🚀 Launch Jupyter: `jupyter notebook` or `jupyter lab`
4. 📖 Open the desired notebook and start learning!
